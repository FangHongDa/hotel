##  项目功能
- 酒店管理系统后台,供管理员管理系统之用
```
客房类型设置
客房设置
员工信息管理
```
- 酒店管理系统系统供酒店所有工作人员使用前台
``` 入住登记：
登记信息
客虎信息
客户订单信息管理
客户入住信息管理
```
- 结账：
```
通过房间号码进行退房结账
```

- 酒店管理系统系统供酒店客户使用前台
```浏览酒店信息
用户管理
预订客房
找回个人密码
调用支付宝接口进行支付
```
